focuspang 서비스 기능 추가

프로젝트 요약

학교 수업을 위해 쓰이는 웹 및 모바일 앱 스위트 focuspang에 대하여 기능을 추가할 것을 의뢰받았습니다. 기능은 크게 모둠 활동의 추가, 선생님 UI의 편의성 개선이었습니다.

모둠 활동 기능 추가

모둠 활동 기능은 다시 선생님 쪽 기능과 학생 쪽 기능으로 나뉩니다.

모듬 활동 기능-선생님

선생님 쪽에서는 다시 다음과 같은 기능을 필요로 했습니다.

  • 모둠 활동이 없는 학급/과정에서 모둠 활동을 새로 편성할 수 있는 기능
  • 모둠 활동이 존재할 때 학생들을 모둠으로 편성할 수 있는 기능
  • 수동 편성
  • 랜덤 편성
  • 편성된 모둠을 수정하는 기능

또한, 모둠 활동은 하나가 아니기 때문에 한 학생이 서로 다른 모둠 활동을 통해 복수의 모둠에 속해 있을 수도 있다는 요구사항도 있었습니다.

모둠 활동 기능-학생

학생 쪽은 조금 더 간단하게, 모둠 활동을 선택하면 같은 모둠에 속한 학생의 화면만을 보는 기능을 필요로 했습니다. 선생님이 학생을 볼 때와 마찬가지로 디바이스 화면을 원격으로 중계해야 하지만, 디바이스 화면을 중계하는 부분은 기술적 이유로 placeholder로 대체하였습니다.

선생님 UI 편의성 개선

기존의 학생 아이콘이 정렬되는 기능에 더해, 모둠 활동과 모둠에 따라 학생을 정렬할 수 있는 기능을 추가하였습니다. 아울러 드래그 앤 드롭으로 정렬 순서를 수동으로 바꿀 수 있고, 정렬 리셋 기능도 구비하였습니다.

구현 방식

구현에 사용할 수 있는 기간이 그렇게 길지 않고 백엔드 개발자가 다른 일로 바빴던 상황이기 때문에 프론트엔드 측에서 자체적으로 표시 전용의 더미 데이터를 만들어 넣는 식으로 구현하였습니다. 더불어 이 로직을 만약에 통합 개발 과정에서 나중에 수정할 경우 더미 데이터를 넣는 코드를 쉽게 실제 백엔드와의 통신 과정으로 변환할 수 있도록 코드의 분리도 어느 정도 해 놓았습니다.

드래그 앤 드롭 기능의 구현은 React.js용으로 드래그 기능이 구현된 오픈 소스 라이브러리를 도입해 활용했습니다.
