진행 내역 요약

웹 개발자를 지망하는 학생이 개발 진행 중인 슈팅 게임을 현재의 JavaScript 환경에 맞게 수정하고 기능 추가를 돕는 작업을 하였습니다. 이 과정에서 저 자신도 현재 쓰이는 JavaScript 상세 (TypeScript 문법이나 키 입력을 받아들이는 방법의 차이, 웹 오디오 관련 심화 프레임워크 등)를 돌아보면서 웹 개발 역량을 늘릴 수 있었습니다.

반면, 게임성 면에서는 피드백을 거의 하지 않고 학생이 원하는 게임성을 지향하도록 존중하였으며, 동작 부분 및 코드의 현대화 면에서만 지도하였습니다.
