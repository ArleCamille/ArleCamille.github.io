OCR 코드 개선 및 모델 인식률 개선

  • 2023년 07월 10일

  • Hyeseon Oh

프로젝트 요약

의뢰인은 반도체 웨이퍼 ID의 인식을 위해 OCR을 사용하고 있었는데, EasyOCR의 일반적인 모델을 사용하고 있었기에 인식률이 그렇게 좋지 않았습니다. 이 점을 개선하기 위해 ID 인식률의 향상을 의뢰받았으며, 다방면에 걸쳐서 개선 작업을 행했습니다.

기본적인 코드의 개선

기존에 작성된 코드는 이미지에 특별한 전처리를 가하지 않고 그대로 사용했는데, 적절하게 이미지를 처리하면 인식률이 어느 정도 올라가는 효과가 있습니다. 이에 Gaussian thresholding을 적용하여 이미지를 흑백의 이미지로 변환한 뒤, 해당 이미지에서 추가로 노이즈를 제거하기 위해 이미지에 closing (dilation 후 erosion)을 가해서 노이즈를 정리한 뒤 erosion (검은 영역을 확장시킴)을 가해 글씨를 조금 더 굵게 했습니다.

인공 트레이닝 데이터 제작

이런 조치는 확실히 정확성 향상에 도움이 되기는 하였으나, 이것만으로 의뢰주가 원하는 정확성을 달성할 수는 없었습니다. 그러나 웨이퍼 ID 자체가 의뢰주 입장에서나 제 입장에서나 상당히 구하기 어려운 데이터였기 때문에 결국 인공적으로 데이터를 만들어야 할 필요성이 있었고, 이를 위해 CRAFT 모델 학습을 위해 사용되던 기존 데이터 생성기를 개조하여 비트맵 폰트에도 사용할 수 있도록 하였고, 이를 통해 약 6000건 내외의 인공 트레이닝 데이터를 제작했습니다.

모델 재학습

마지막으로, 인공 데이터 3:실제 데이터 1 비율로 모델 재학습을 진행하였습니다. 결과적으로 의뢰주가 요구한 상황에서 97% 이상의 높은 정확성을 보여주었습니다.
