Android용 CAD 대응 펜 마우스 드로잉 앱 개발

  • 2023년 07월 10일
  • Contracted Projects
  • Hyeseon Oh
  • #project, #contracted

프로젝트 요약

차량 도장 작업에서 차량의 모양을 따라 2차원상에서 실측 규모로 스케치를 할 일이 있는 클라이언트의 의뢰로 개발하였습니다. 의뢰받을 때 요구받은 부분은 크게 다음과 같았습니다. - 기본적인 드로잉 기능 - 펜 기능 (직선, 다각형, 자유 곡선) - 텍스트 입력 및 삽입 - 도형 위 및 자유로운 장소에 점 삽입 가능, 반대로 점에서 도형 시작 가능 - 색상 변경 가능 - 터치를 이용한 기본적인 내비게이션 기능 - 선택을 위시한 편집 기능 - 내부 저장장치에 저장이 가능해야 함 - 내부 형식으로 저장 및 CAD로 내보내기가 가능해야 함 - CAD 프로그램으로 내보내는 과정에서 색상 등의 정보가 손실되면 안 됨

세부 구현

이 요구사항에 대응하기 위해, 다음과 같이 작업하였습니다. - CAD 프로그램 호환을 염두에 두어, 내부 형식 구조를 DXF에 대응하도록 작업 - 도형은 Block 단위로 저장 - 하나의 Block에 여러 도형이 있을 수 있으나, 점 등 이어지는 부분이 있을 경우에만 하나의 Block에 있도록 하였음 - 펜 마우스의 움직임과 터치 움직임을 따로 취급 - 펜 마우스의 움직임은 포인팅 디바이스로서 독립된 입력 핸들러에 들어감 - 터치 움직임은 pinch to zoom 등의 멀티터치 핸들링이 가능하면서 포인팅 디바이스의 움직임에 구애받지 않음 - 기본적인 드로잉 툴의 구현을 위해 메모리 내부에 다각형, 점 및 텍스트 자료 구조를 도입 - DXF로 내보내기를 한다고는 하나 최종적으로는 CAD 프로그램에서 마무리를 진행하기 때문에 구현 및 연산에 코스트가 들어가는 커브 형태보다는 마우스의 움직임을 그대로 다각형으로 처리하기로 함 - 점과 텍스트를 구현하기 위한 독립된 구조체를 사용 - 기존 오픈 소스 라이브러리와 연계하여 DXF 파일로 내보내기를 구현 - React Native 앱을 브릿지로 사용하여 해당 라이브러리를 호출

이외에도 현재 유지보수 단계에서 다른 편의기능의 추가가 예정되어 있습니다.
